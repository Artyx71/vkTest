# Movie Browser SPA

Одностраничное приложение (SPA) для поиска и просмотра информации о фильмах. Написано на React, TypeScript и использует Kinopoisk Unofficial API.

## Функционал
- **Главная страница**: Бесконечный скролл списка фильмов
- **Фильтрация**: По жанру, году, рейтингу и типу
- **Панель сравнения**: Выбор двух и более фильмов для показа сравнительной таблицы (включается кнопкой "⚖" на карточках)
- **Страница фильма**: Подробная информация (постер, рейтинги, описание и т.д.)
- **Избранное**: Страница `/favorites`, данные сохраняются локально после перезагрузки
- **Адаптивность**: Мобильная версия с responsive дизайном

## Стек технологий
- React 18
- TypeScript
- Vite
- Zustand (Управление состоянием фильтров, избранных и сравнения)
- React Router v6 (Роутинг)
- CSS Modules (Изолированная стилизация без крупных библиотек)
- Kinopoisk API Unofficial

## Установка и запуск

1. Склонируйте репозиторий и перейдите в папку проекта.
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Создайте файл `.env` в корне проекта (вы можете скопировать `.env.example`):
   ```bash
   cp .env.example .env
   ```
4. Вставьте ваш токен API Кинопоиска в файл `.env`:
   ```
   VITE_KINOPOISK_API_KEY=8c8e1a9f-6d5d-4f83-b9ed-112233445566
   ```
5. Запустите в режиме разработчика:
   ```bash
   npm run dev
   ```
   Приложение откроется на `http://localhost:5173/`

## Сборка для production

```bash
npm run build
```
Готовые статические файлы будут лежать в папке `dist/`.
